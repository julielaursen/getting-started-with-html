HTML Lesson 1 Notes: The Basics of the Web and HTML

The World Wide Web is made up of HTML documents. Word, PDF, images, videos, music and plain text can be displayed and linked to.

HTTP is the protocol that unites the web. HTTP makes requests to servers using your browsers. Browser types:

  • Firefox
  • Internet Explorer
  • Chrome

    • HTML includes markup, which is made up of tags. Tags are structured with contents in between opening and closing tags. One tag is the italicize tag. If you leave a closing italicize(em) tag off everything after the opening tag is italicized HTML attributes set a tag equal to a value in quotes and tags can have multiple attributes.
